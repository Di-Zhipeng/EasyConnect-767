# EasyConnect-767

Make SangFor EasyConnect run on your Arch Linux distribution.

## Usage

```shell
git clone https://github.com/alk3p/EasyConnect-767.git
cd EasyConnect-767 && makepkg
sudo pacman -U easyconnect-7.6.7.3-4-x86_64.pkg.tar.xz
```
